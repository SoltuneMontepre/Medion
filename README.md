### Prerequisite

For frontend:

- [NodeJS 18+](https://nodejs.org/en/download)
- [Angular CLI](https://angular.dev/installation)

For backend:

- [.NET 9.0](https://dotnet.microsoft.com/en-us/download/dotnet/9.0) (SDK & Runtime) - required for building and running backend services. Verify with `dotnet --version`.
- [Docker 23+](https://docs.docker.com/desktop/) (Engine/CLI) - required for containerized workflows. Verify with `docker --version`.

### Developement quickstart

1. Clone this repository with git:

```cmd
git clone https://github.com/SoltuneMontepre/Medion.git

cd Medion
```
